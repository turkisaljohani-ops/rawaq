# 🎯 رواق — لعبة الأسئلة الجماعية

> لعبة تفاعلية جماعية مشابهة لـ Kahoot، تعمل عبر المتصفح بالكامل.

---

## 🚀 تشغيل المشروع

```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل الخادم
npm start
# أو للتطوير مع إعادة التشغيل التلقائي:
npm run dev

# 3. افتح المتصفح على
http://localhost:3000
```

---

## 📁 بنية المشروع

```
rawaq/
├── server.js          ← الخادم (Node.js + Socket.io)
├── package.json       ← تبعيات المشروع
└── public/
    ├── index.html     ← هيكل الصفحة (كل الشاشات)
    ├── style.css      ← التصميم الكامل
    └── client.js      ← منطق العميل والاتصال
```

---

## 🗂️ شرح كل ملف

### `server.js`
الخادم الرئيسي. يدير:
- إنشاء الغرف وتوليد الـ PIN
- إدارة اللاعبين والنقاط
- تزامن الأسئلة والمؤقتات عبر Socket.io
- منطق انتهاء اللعبة والـ Leaderboard

### `public/index.html`
يحتوي على كل شاشات اللعبة:
- الرئيسية، إنشاء غرفة، انضمام
- Lobby، سؤال المضيف، سؤال اللاعب
- Leaderboard، نهاية اللعبة

### `public/style.css`
تصميم كامل بـ CSS Variables، داكن وعصري، متجاوب للجوال.

### `public/client.js`
منطق الواجهة:
- الاتصال بـ Socket.io وإعادة الاتصال التلقائية
- إدارة الشاشات، المؤقتات، التأثيرات
- معالجة أحداث اللاعب والمضيف

---

## 🎮 كيفية اللعب

1. **المضيف** يفتح الموقع → "أنشئ غرفة" → يحصل على PIN
2. **اللاعبون** يدخلون الـ PIN أو يفتحون رابط المشاركة
3. المضيف يضغط "ابدأ اللعبة"
4. كل لاعب يرى 4 أزرار ملونة على شاشته
5. بعد كل سؤال يظهر الـ Leaderboard
6. في النهاية تظهر منصة الفائزين 🏆

---

## ⚙️ تخصيص الأسئلة

في `server.js`، عدّل مصفوفة `DEFAULT_QUESTIONS`:

```js
{
  id: 1,
  text: 'سؤالك هنا؟',
  options: ['خيار1', 'خيار2', 'خيار3', 'خيار4'],
  correct: 0,   // رقم الإجابة الصحيحة (0-3)
  time: 15      // ثوانٍ
}
```
